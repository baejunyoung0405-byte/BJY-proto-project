# TodoWpfApp

Windows desktop todo app built with WPF + MVVM + SQLite.

## Run

1. Install .NET 8 SDK (Windows x64)
2. In this folder:
   - `dotnet restore`
   - `dotnet run`

## Features

- Add, update, delete tasks
- Toggle completion
- Due date, priority, notes
- Filters: All, Today, This Week, Completed
- System tray support (minimize to tray)
- Reminder balloon notifications for overdue / due-by-tomorrow tasks
- SQLite local persistence
